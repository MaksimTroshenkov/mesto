## Что тут есть:
* Адптивная вёрстка.
* Все ссылки и интерактивные элементы имеют состояние наведения.
* Попапы плавно открываются и закрываются.
* Редактирование текста профиля.
* Добавление и удаление карточек с фотографиями.
* Карточки можно "лайкать".
* Методология БЭМ.
